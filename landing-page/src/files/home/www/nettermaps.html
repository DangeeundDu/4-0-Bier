nettermaps

This website shows you popular places on an interactive map. Browse through the world to find the most interesting spots and mark them with a click. You can name the marker to clarify what can be seen there. Furthermore you may publish your marker, so everybody can profit from it.

Functionality

The following functionality must be retained:

  • It must be possible to add markers. Said markers can be public or private (this option has to be retained).
  • It must be possible to view markers on the map.
  • The HTTP interface (so the parameter names etc.) of db.php must not be changed.
