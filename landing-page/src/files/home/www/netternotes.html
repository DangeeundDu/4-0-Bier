netternotes

The netterverse also provides you with the most advanced note taking service. You can create notes, lock them if you want to and also search for your notes.

Functionality

The following functions are supported by this service and must be retained:

  • Create (public) unlocked notes
  • Create (private) locked notes
  • View notes
  • Search for notes
